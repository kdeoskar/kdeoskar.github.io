Maps of Constant Rank

  • Essentially the only property which characterizes a linear maps, if we are allowed to choose separate coordinates for the domain and range, is the rank of the map i.e. dim⁡Im(F)\dim \mathrm{Im} (F)dimIm(F).

  • We define the rank of a smooth map F:Mm→NnF : M^m \rightarrow N^nF:Mm→Nn between manifolds at point p∈Mp \in Mp∈M to be the rank of the differential differential dFp:TpM→TpNdF_p : T_p M \rightarrow T_p NdFp​:Tp​M→Tp​N.

  • The rank of a linear map is never higher than the dimensions of its domain and range, so if a map F:Mm→NnF : M^m \rightarrow N^nF:Mm→Nn has rank equal to min⁡{m,n}\min\{m,n\}min{m,n} at point ppp, we say it has full rank at p, and if it has full rank at all points in MMM, we say F has full rank.
